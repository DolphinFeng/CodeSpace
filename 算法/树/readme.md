# 二叉树
    只有两个枝丫，或者没有分叉
    - 数据域
    - 左子树
    - 右子树

    树的遍历：从上到下，从下到上
        A
      B   C
    D  E F  G
满二叉树：每个节点都有两个
1. 空树或者每个节点必须有左子树和右子树
- 先序(前序)遍历(递归) 根左右  ABDECFG
- 中序遍历(递归) 左根右   看见树就秉承这个规则  DBEAFCG
- 后序遍历(递归) 左右根   DEBFGCA

- 层次遍历(迭代)
