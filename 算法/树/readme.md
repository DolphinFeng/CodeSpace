# 二叉树
    只有两个枝丫，或者没有分叉
    - 数据域
    - 左子树
    - 右子树

    树的遍历：从上到下，从下到上
        A
      B   C
    D  E F  G
满二叉树：每个节点都有两个
1. 空树或者每个节点必须有左子树和右子树
- 先序(前序)遍历(递归) 根左右  ABDECFG
- 中序遍历(递归) 左根右   看见树就秉承这个规则  DBEAFCG
- 后序遍历(递归) 左右根   DEBFGCA

- 层次遍历(迭代) ABCDEFG 借助队列  只能迭代

# 探索树的思想：深度优先 & 广度优先
    这两个会出场景题。eg.迷宫-深度优先
    - 深度优先（Depth First Search DFS）
        将一条路走到底，再走另一条
        本质：栈
        二叉树的前中后序遍历都是深度优先
    - 广度优先（breadth First Search BFS）
        先将第一层的路走完
        二叉树的层次遍历就是广度优先
