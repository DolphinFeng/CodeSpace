# 链表
    也是一种数据结构（增删改限定到某种规则就是一种数据结构）
    链表也有有序，线性结构，问题在于，他只有一个前驱节点，一个后驱节点

    - 节点：链表的元素
        链表中，节点允许散落在内存中任意地方，离散型
        js中链表也没有特殊符号，用对象花括号模拟链表

    增删  数组o(n)  链表o(1)   读取  数组o(1)  链表o(n)
